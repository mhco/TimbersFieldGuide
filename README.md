Raid Invites
- Maintain a list of characters on a "roster"
    - Two rosters: raid team roster (permanent), and event roster (temporary)
    - Rosters are saved to the character you're using the addon with
- Import a list of players to create a roster
    - Somehow get the list of characters from Discord's event app, even if it's manual
- Export raid team roster
    - Allow sharing of raid invite duties if you go out of town or quit the game
- Flag characters in the roster to automatically give them special roles
    - Designate a master looter for a given roster
        - When this person joins the raid, automatically give them master looter capacity
    - Designate assistants
        - When they join the raid, automatically give them raid assist
    - Designate tanks
        - When they join the raid, automatically mark them as a tank
